About the Project

This web app fetches and displays the most popular YouTube videos by region using the YouTube API. Users can select a country and view trending videos.

🔹 Key Features:

  • 🌍 View trending videos from different regions
  • 🔎 Click on videos to watch directly on YouTube
  • 📊 Displays view count & channel name
  • 📱 Responsive design for mobile and desktop
